## Technical Details

### Architecture
- **Backend:** Flask (Python)
- **Frontend:** Vanilla JavaScript
- **Processing:** Background threads
- **Updates:** Auto-polling (2 second interval)
- **Storage:** Local filesystem

### API Endpoints
- `POST /api/upload` - Upload files
- `GET /api/jobs` - List all jobs
- `GET /api/jobs/<id>` - Get job status
- `GET /api/download/<id>` - Download result
- `GET /api/stats` - Get statistics
- `GET /health` - Health check

### Code Stats
- **Web App:** `src/web_app.py` (237 lines)
- **Frontend HTML:** `web/templates/index.html`
- **CSS:** `web/static/css/styles.css` (400+ lines)
- **JavaScript:** `web/static/js/app.js` (200+ lines)

## Installation

Already have v1.2? Just rebuild:

```bash
# Pull latest code
git pull

# Rebuild web container
docker-compose build web

# Start web UI
docker-compose up -d web
```

New install:

```bash
# Clone repo
git clone [repo-url]
cd myfiles_to_markdown

# Start Ollama (if not running)
ollama serve

# Pull AI models
ollama pull llama3.2:latest
ollama pull llava:7b

# Build and start
docker-compose build
docker-compose up -d web

# Open browser
open http://localhost:3000
```
